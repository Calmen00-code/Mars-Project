This is a program that simulates the robot rover activity on Mars.
There is no User Interface provided by this program as it will not be used by human on earth

Instead, the simulation is done by storing all command in a list of string and then the
program will read every single command in the infinite loop.
-------------------------------------------------------------------------------------------------

To compile the program,
    In the terminal type "ant" then hit Enter

To run the program,
    In the terminal type "java -jar ./dist/Assignment2.jar"

The program will be running and displaying the feedback by its command passed as it progress.
